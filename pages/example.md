# 안녕하세요! 👋

이 블로그에 오신 것을 환영합니다. 이 글은 샘플 게시글로, 블로그의 다양한 기능을 보여드립니다.

## 마크다운 기능 소개

이 블로그는 **마크다운**을 지원합니다. 다양한 형식의 글을 작성할 수 있어요.

### 텍스트 스타일링

- **굵은 글씨**는 `**텍스트**`로 작성합니다.
- *기울임 글씨*는 `*텍스트*`로 작성합니다.
- ~~취소선~~은 `~~텍스트~~`로 작성합니다.
- `인라인 코드`는 백틱으로 감싸면 됩니다.

### 링크와 이미지

[GitHub](https://github.com)처럼 링크를 삽입할 수 있습니다.

### 인용문

> 인용문은 `>`로 시작합니다.
> 여러 줄로 작성할 수 있어요.

### 코드 블록

JavaScript 코드 예시입니다:

```javascript
// 간단한 함수 예시
function greet(name) {
  return `안녕하세요, ${name}님!`;
}

console.log(greet('방문자'));
```

Python 코드도 지원합니다:

```python
def fibonacci(n):
    """피보나치 수열 생성"""
    if n <= 1:
        return n
    return fibonacci(n - 1) + fibonacci(n - 2)

# 처음 10개 출력
for i in range(10):
    print(fibonacci(i))
```

## 블로그 사용법

### 새 글 작성하기

1. `pages/` 폴더에 `.md` 파일을 생성합니다.
2. 파일 상단에 Front Matter를 작성합니다:

```yaml
---
title: '게시글 제목'
date: 2025-01-01
tags: ['태그1', '태그2']
category: '카테고리명'
description: '게시글 설명'
---
```

3. 마크다운으로 본문을 작성합니다.
4. Git에 커밋하고 push하면 자동으로 배포됩니다!

### 다크/라이트 모드

우측 상단의 🌙/☀️ 버튼을 클릭하면 테마를 전환할 수 있습니다. 설정은 브라우저에 저장되어 다음 방문 시에도 유지됩니다.

### 검색 기능

메인 페이지의 검색창에서 게시글을 검색할 수 있습니다. 제목, 내용, 태그, 카테고리 모두 검색 대상입니다.

**단축키**: `Ctrl + K` (Windows) 또는 `Cmd + K` (Mac)로 검색창에 빠르게 포커스할 수 있습니다.

### 댓글 기능

각 게시글 하단에서 GitHub 계정으로 댓글을 남길 수 있습니다. Giscus를 사용하여 GitHub Discussions와 연동됩니다.
